# Web Element Selector v1.6.0

This release adds a proper settings page so the extension can be personalized without touching source files.

## Highlights

- New `Options` page for default launch behavior
- Configure default Target AI, Safe/Full mode, Focused/Nearby mode, and default task text
- Popup now links directly to settings
- Extension-level defaults are applied automatically when Selector launches on a page

## Install

### Unpacked extension

1. Open `chrome://extensions` or `edge://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the `web-element-selector` folder
5. Open the extension popup and click `Open settings`

### Bookmarklet

Open `https://kenchikuliu.github.io/web-element-selector/` and drag the bookmark to your bookmarks bar.

### Packaged zip

Use the attached `web-element-selector-v1.6.0.zip` asset for manual extraction and sharing.
