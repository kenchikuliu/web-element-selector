# Web Element Selector v2.0.0

This release adds a minimum viable Chrome Web Store submission kit so the project can move from a polished GitHub repo to a store-ready listing.

## Highlights

- New `docs/chrome-web-store/` directory with listing copy
- Privacy and permissions wording aligned to the current extension behavior
- Screenshot plan and caption guidance for store upload
- Minimal submission checklist for store prep

## Included docs

- `docs/chrome-web-store/listing.md`
- `docs/chrome-web-store/privacy.md`
- `docs/chrome-web-store/screenshots.md`
- `docs/chrome-web-store/submission-checklist.md`
