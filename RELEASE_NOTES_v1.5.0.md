# Web Element Selector v1.5.0

This release upgrades the popup from a simple launcher into a more complete control surface.

## Highlights

- Last-launch activity is saved and shown in the popup
- Built-in usage steps reduce guesswork for first-time users
- Shortcut cards surface the main keyboard controls directly in the extension
- Better restricted-page messaging when injection is unavailable

## Install

### Unpacked extension

1. Open `chrome://extensions` or `edge://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the `web-element-selector` folder
5. Click the extension icon to open the popup
6. Press `Launch Selector`

### Bookmarklet

Open `https://kenchikuliu.github.io/web-element-selector/` and drag the bookmark to your bookmarks bar.

### Packaged zip

Use the attached `web-element-selector-v1.5.0.zip` asset for manual extraction and sharing.
