# Web Element Selector v1.9.0

This release finishes the public presentation layer with a proper social preview card for shared links.

## Highlights

- Added a dedicated 1200x630 social preview image
- Install page now includes Open Graph and Twitter metadata
- Shared links to the GitHub Pages install page can render with a large preview image
- README now surfaces the preview asset directly in the repository homepage

## Install

### Unpacked extension

1. Open `chrome://extensions` or `edge://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the `web-element-selector` folder
5. Open the popup and press `Launch Selector`

### Bookmarklet

Open `https://kenchikuliu.github.io/web-element-selector/` and drag the bookmark to your bookmarks bar.

### Packaged zip

Use the attached `web-element-selector-v1.9.0.zip` asset for manual extraction and sharing.
