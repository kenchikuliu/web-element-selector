## Detailed description

### Recommended version

`Web Element Selector helps you turn vague UI feedback into precise implementation prompts.

Launch it on any page, click the exact region you want to change, describe the improvement, and copy a structured prompt for Codex, Claude Code, Cursor, or another AI coding assistant.

Use it on local development pages, staging URLs, production pages, and bookmarklet-compatible sites.

Core capabilities:

- Select one or multiple page regions
- Add a task for the selected area
- Add per-element notes when needed
- Export prompts for Codex, Claude Code, Cursor, JSON, or selector-only workflows
- Switch between Safe and Full export modes
- Use Focused or Nearby context depending on how much layout context is needed
- Launch from a popup and configure default behavior from settings

Web Element Selector is designed for designers, frontend engineers, and AI-assisted builders who want cleaner handoff between the browser and their coding tool.

Privacy note:

Safe mode is the default. It omits visible text, HTML, and data attributes unless you explicitly switch to Full mode.`
