# Chrome Web Store Kit

This folder contains the minimum reusable material needed to submit `Web Element Selector` to the Chrome Web Store.

## Files

- `listing.md`: ready-to-use listing copy
- `privacy.md`: privacy and permissions explanation
- `screenshots.md`: screenshot order, captions, and asset guidance
- `submission-checklist.md`: pre-submit checklist for the store flow
- `assets/`: place exported store screenshots and promo graphics here

## Recommended order

1. Read `listing.md`
2. Upload screenshots in the order from `screenshots.md`
3. Reuse the privacy wording from `privacy.md`
4. Walk through `submission-checklist.md` before hitting submit
